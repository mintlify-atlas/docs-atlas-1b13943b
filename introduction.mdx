## Welcome to Redirect Trace

Redirect Trace is a professional URL redirect tracer for Raycast that analyzes redirect chains, removes tracking parameters, and helps you debug links. Like wheregoes.com for Raycast, but faster and more powerful.

## Use cases

Redirect Trace helps with multiple workflows:

### Security

Detect phishing attempts, prevent malware, and verify legitimate vs malicious redirects before clicking unknown links.

### Marketing

Analyze campaign redirect paths, find unnecessary redirects slowing down links, and inspect UTM parameters to see what tracking data is being collected.

### Development

Test redirect behavior in applications, check for redirect chains affecting SEO, and identify slow or broken redirects during debugging.

## What you'll see

When you trace a URL, Redirect Trace provides:

<Steps>
  <Step title="Summary section">
    View the original URL with character count, final destination, clean URL with tracking removed, and total redirect count
  </Step>
  <Step title="Redirect chain">
    Inspect each step showing HTTP status code, status message, URL at each step, response headers, and step number
  </Step>
  <Step title="Clean URLs">
    Get the final URL with all tracking parameters automatically removed
  </Step>
</Steps>

<Note>
  All URL analysis happens locally on your device. Redirect Trace does not store, log, or track any URLs you process.
</Note>
